1. Godkännande av villkor

Genom att ladda ner, installera eller använda Gradual ("appen") godkänner du dessa användarvillkor ("villkoren"). Dessa villkor kompletterar Apples licensavtal för slutanvändare av licensierade applikationer, som reglerar din licens att använda appen. Om du inte godkänner dessa villkor, använd inte appen.

2. Beskrivning av tjänsten

Gradual är en personlig hälsospårningsapplikation som aggregerar hälsodata från flera källor, inklusive Apple HealthKit, Strava och manuellt inmatade labbrapporter. Appen beräknar ett sammansatt Gradual Age och hjälper dig att spåra tillskottsprotokoll, biomarkörer och träningsmått.

3. Inte medicinsk rådgivning

Viktigt: Appen är utformad enbart för informations- och hälsospårningssyften. Inget i appen utgör medicinsk rådgivning, diagnos eller behandling.

Gradual Age, biomarkörsanalys och all annan presenterad data är inte substitut för professionell medicinsk rådgivning. Rådfråga alltid en kvalificerad vårdgivare innan du fattar beslut om din hälsa, mediciner eller tillskottsprotokoll.

4. Användarkonton och data

Du ansvarar för riktigheten av all data du matar in i appen, inklusive labbresultat och tillskottsinformation. Dina hälsodata lagras lokalt på din enhet med krypterad lagring (expo-secure-store) och synkroniseras till ditt personliga iCloud-konto via Apples CloudKit. Du kan exportera eller radera dina data när som helst från profilskärmen.

5. Tredjepartstjänster

Appen integreras med följande tredjepartstjänster. Din användning av dessa integrationer omfattas av respektive tjänsts egna villkor:

  • Apple HealthKit — Hälso- och träningsdata läses med ditt uttryckliga tillstånd. Dessa data synkroniseras via ditt personliga iCloud-konto och lagras inte på Lawton Labs servrar.
  • Strava — Aktivitetsdata nås via OAuth 2.0 med PKCE-autentisering. Endast aktivitetsdata som du godkänner hämtas.
  • OpenAI — När du importerar en labbrapport och väljer AI-analys skickas din labbrapport-PDF till OpenAI:s API för AI-driven biomarkörextraktion. Se vår integritetspolicy för detaljer om hur dessa data hanteras.

6. Bearbetning av labbrapporter

När du importerar en labbrapport använder appen två metoder för att extrahera biomarkördata:

  • Lokal analys — En regex-baserad parser körs helt på din enhet utan extern dataöverföring.
  • AI-driven analys — Din labbrapport-PDF skickas till OpenAI:s API för mer exakt extraktion. OpenAI bearbetar dessa data enligt sina egna API-användarvillkor och integritetspolicy. Den lokala parsern finns tillgänglig som reserv om AI-analysen misslyckas.

När du importerar en labbrapport-PDF visas en samtyckesdialog innan några data skickas. Om du inte vill dela labbdata med OpenAI kan du istället använda alternativet "Klistra in resultat", som analyserar helt på enheten.
