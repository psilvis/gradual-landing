Gradual

Integritetspolicy

Senast uppdaterad: 26 februari 2025

1. Översikt

Gradual ("appen") är byggd med ett integritetsfokuserat tillvägagångssätt. Dina hälsodata lagras säkert på din enhet och synkroniseras via Apples iCloud-tjänst med CloudKit. Dina data lagras aldrig på Lawton Labs servrar. Vissa funktioner använder tredjepartstjänster enligt beskrivningen nedan.

2. Data vi samlar in

Appen samlar in och lagrar följande data lokalt på din enhet:

  • Profilinformation — Ditt namn och dina inställningar som du anger under introduktionen.
  • Hälsomått — Data från Apple HealthKit (steg, hjärtfrekvens, HRV, sömn, VO2max och andra mått) med ditt uttryckliga tillstånd.
  • Aktivitetsdata — Tränings- och aktivitetsinformation från Strava.
  • Labbresultat — Biomarkörsvärden som extraherats från labbrapporter du importerar.
  • Tillskottsprotokoll — Kosttillskott, doseringar och följsamhetsloggar som du skapar.
  • Humörcheckar — Dagliga humör- och stressbedömningar som du registrerar.

Allt ovanstående krypteras på enheten och synkroniseras säkert via Apple iCloud (CloudKit). Dina data lagras i ditt personliga iCloud-konto och lagras aldrig på Lawton Labs servrar.

3. Data som delas med tredje part

OpenAI (bearbetning av labbrapporter)

När du använder labbimportfunktionen och väljer AI-analys skickas din labbrapport-PDF till OpenAI:s API för AI-driven biomarkörextraktion. Det innebär:

  • Din labbrapport-PDF — som kan innehålla ditt namn, biomarkörsvärden, referensintervall, labbnamn och testdatum — överförs till OpenAI:s servrar för bearbetning.
  • OpenAI bearbetar dessa data i enlighet med OpenAI:s API-användarvillkor och integritetspolicy.
  • Enligt OpenAI:s policy för API-dataanvändning används data som skickas via API:et inte för att träna OpenAI:s modeller.
  • Data överförs via krypterad HTTPS-anslutning.
  • Inga enhetsidentifierare eller personlig information utöver rapportinnehållet överförs.
  • Om AI-analysen misslyckas faller appen tillbaka till en helt lokal regex-baserad parser som inte överför några data.

Du kan undvika att dela labbdata med OpenAI genom att inte använda PDF-labbimportfunktionen. Alternativet "Klistra in resultat" använder en parser på enheten som inte överför några data.

4. Apple HealthKit-data

Hälsodata som nås via Apple HealthKit hanteras enligt Apples HealthKit-riktlinjer:

  • Data nås först efter att du har gett uttryckligt tillstånd.
  • HealthKit-data läses enbart — appen skriver inte data till HealthKit.
  • HealthKit-data synkroniseras via ditt personliga iCloud-konto med Apples CloudKit. Dessa data lagras i din privata iCloud-container och lagras aldrig på Lawton Labs servrar eller överförs till tredje part.
  • HealthKit-data används aldrig i reklam- eller marknadsföringssyfte.

5. Strava-integration

När du ansluter ditt Strava-konto:

  • Autentisering sker via OAuth 2.0 med PKCE — appen ser eller lagrar aldrig ditt Strava-lösenord.
  • Endast aktivitetsdata (typ, varaktighet, distans, hjärtfrekvens) hämtas från Stravas API.
  • Hämtad aktivitetsdata cachelagras lokalt på din enhet.
  • Du kan koppla bort Strava när som helst från profilskärmen.

6. Datalagring och säkerhet

  • Känsliga data (API-nycklar, token, hälsodata) lagras med expo-secure-store, som använder iOS-nyckelringen för krypterad lagring.
  • Hälsomått som lagras i SQLite krypteras med AES-256-GCM på applikationsnivå. Krypteringsnyckeln lagras i iOS-nyckelringen.
  • Dina data synkroniseras till ditt personliga iCloud-konto via Apples CloudKit. Dessa data lagras i en privat CloudKit-container som bara du har tillgång till. Inga hälsodata lagras på servrar som drivs av Lawton Labs.
  • Dataexport finns tillgänglig i JSON-format från profilskärmen.

7. Datalagring och radering

  • All data lagras på din enhet och synkroniseras till ditt personliga iCloud-konto. Data finns kvar tills du raderar den.
  • Du kan radera all data när som helst med alternativet "Radera all data" på profilskärmen. Detta tar bort data från din enhet och iCloud.
  • Avinstallation av appen tar bort all lokalt lagrad data. Data i ditt iCloud-konto kan finnas kvar tills den raderas via appen eller iCloud-inställningarna.
  • Data som skickas till OpenAI för labbanalys bearbetas tillfälligt och behålls inte av OpenAI för modellträning, enligt OpenAI:s policy för API-dataanvändning.

8. Barns integritet

Appen är inte avsedd för användning av personer under 13 år. Vi samlar inte medvetet in personuppgifter från barn under 13 år.

9. Analys och felspårning

Appen använder följande tjänster för analys och felspårning:

  • Amplitude — Anonym användningsanalys för att hjälpa oss förstå hur funktioner används och förbättra appen. Inga hälsodata eller personligt identifierbar information skickas till Amplitude.
  • Sentry — Krasch- och felrapportering för att hjälpa oss identifiera och åtgärda buggar. Felrapporter kan innehålla enhetstyp, OS-version och stack traces, men innehåller inte dina hälsodata.

Ingen av dessa tjänster tar emot dina hälsodata, biomarkörer eller personligt identifierbar hälsoinformation.

10. Ändringar av denna policy

Vi kan uppdatera denna integritetspolicy från tid till annan. Vi meddelar dig om väsentliga ändringar via appen. Fortsatt användning av appen efter ändringar utgör godkännande av den uppdaterade policyn.

11. Dina rättigheter

Du har rätt att:

  • Få tillgång till all din data som lagras i appen.
  • Exportera dina data i ett portabelt format (JSON).
  • Radera all din data när som helst.
  • Koppla bort alla tredjepartstjänster när som helst.
  • Välja att inte använda funktioner som involverar tredjepartsbearbetning av data (labbimport).
