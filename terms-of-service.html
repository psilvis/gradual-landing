The App integrates with the following third-party services. Your use of these integrations is subject to each service's own terms:

6. Lab Report Processing

When you import a lab report, the App uses two methods to extract biomarker data:

  • Local parsing — A regex-based parser runs entirely on your device with no external data transmission.
  • AI-powered parsing — Your lab report PDF is sent to OpenAI's API for more accurate extraction. OpenAI processes this data according to its own API Terms of Use and Privacy Policy. The local parser is available as a fallback if AI parsing fails.

When importing a lab report PDF, you are presented with a consent dialog before any data is sent. If you do not wish to share lab data with OpenAI, you can use the "Paste Results" option instead, which parses entirely on-device.
