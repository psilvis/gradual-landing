2. Beschreibung des Dienstes

Gradual ist eine persönliche Gesundheits-Tracking-Anwendung, die Gesundheitsdaten aus mehreren Quellen aggregiert, darunter Apple HealthKit, Strava und manuell eingegebene Laborberichte. Die App berechnet ein zusammengesetztes Gradual Age und hilft Ihnen, Nahrungsergänzungsprotokolle, Biomarker und Fitnessmetriken zu verfolgen.

3. Kein medizinischer Rat

Wichtig: Die App dient ausschließlich zu Informations- und Wellness-Tracking-Zwecken. Nichts in der App stellt eine medizinische Beratung, Diagnose oder Behandlung dar.

Das Gradual Age, die Biomarkeranalyse und alle anderen dargestellten Daten sind kein Ersatz für eine professionelle medizinische Beratung. Konsultieren Sie immer einen qualifizierten Gesundheitsdienstleister, bevor Sie Entscheidungen über Ihre Gesundheit, Medikamente oder Nahrungsergänzungsprotokolle treffen.

4. Benutzerkonten und Daten

Sie sind für die Richtigkeit aller Daten verantwortlich, die Sie in die App eingeben, einschließlich Laborergebnisse und Informationen zu Nahrungsergänzungsmitteln. Ihre Gesundheitsdaten werden lokal auf Ihrem Gerät mittels verschlüsselter Speicherung (expo-secure-store) gespeichert und über Apples CloudKit mit Ihrem persönlichen iCloud-Konto synchronisiert. Sie können Ihre Daten jederzeit über den Profilbildschirm exportieren oder löschen.

5. Drittanbieterdienste

Die App integriert sich mit den folgenden Drittanbieterdiensten. Ihre Nutzung dieser Integrationen unterliegt den jeweiligen Bedingungen des jeweiligen Dienstes:

  • Apple HealthKit — Gesundheits- und Fitnessdaten werden mit Ihrer ausdrücklichen Genehmigung gelesen. Diese Daten werden über Ihr persönliches iCloud-Konto synchronisiert und nicht auf Servern von Lawton Labs gespeichert.
  • Strava — Auf Aktivitätsdaten wird über OAuth 2.0 mit PKCE-Authentifizierung zugegriffen. Es werden nur die von Ihnen autorisierten Aktivitätsdaten abgerufen.
  • OpenAI — Wenn Sie einen Laborbericht importieren und die KI-Analyse wählen, wird Ihr Laborbericht-PDF an die API von OpenAI zur KI-gestützten Biomarkerextraktion gesendet. Details zum Umgang mit diesen Daten finden Sie in unserer Datenschutzrichtlinie.

6. Laborberichtverarbeitung

Wenn Sie einen Laborbericht importieren, verwendet die App zwei Methoden zur Extraktion von Biomarkerdaten:

  • Lokale Analyse — Ein regex-basierter Parser läuft vollständig auf Ihrem Gerät ohne externe Datenübertragung.
  • KI-gestützte Analyse — Ihr Laborbericht-PDF wird an die API von OpenAI zur genaueren Extraktion gesendet. OpenAI verarbeitet diese Daten gemäß den eigenen API-Nutzungsbedingungen und der Datenschutzrichtlinie. Der lokale Parser steht als Fallback zur Verfügung, falls die KI-Analyse fehlschlägt.

Beim Importieren eines Laborbericht-PDFs wird Ihnen ein Einwilligungsdialog angezeigt, bevor Daten gesendet werden. Wenn Sie keine Labordaten mit OpenAI teilen möchten, können Sie stattdessen die Option „Ergebnisse einfügen" verwenden, die vollständig auf dem Gerät analysiert.
