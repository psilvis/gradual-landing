Gradual

Datenschutzrichtlinie

Zuletzt aktualisiert: 26. Februar 2025

1. Überblick

Gradual („die App") wurde mit einem datenschutzorientierten Ansatz entwickelt. Ihre Gesundheitsdaten werden sicher auf Ihrem Gerät gespeichert und über Apples iCloud-Dienst mittels CloudKit synchronisiert. Ihre Daten werden niemals auf Servern von Lawton Labs gespeichert. Bestimmte Funktionen nutzen Drittanbieterdienste, wie nachfolgend beschrieben.

2. Daten, die wir erheben

Die App erhebt und speichert die folgenden Daten lokal auf Ihrem Gerät:

  • Profilinformationen — Ihr Name und Ihre Einstellungen, die Sie während des Onboardings eingeben.
  • Gesundheitsmetriken — Daten aus Apple HealthKit (Schritte, Herzfrequenz, HRV, Schlaf, VO2max und weitere Metriken) mit Ihrer ausdrücklichen Genehmigung.
  • Aktivitätsdaten — Trainings- und Aktivitätsinformationen von Strava.
  • Laborergebnisse — Biomarkerwerte, die aus von Ihnen importierten Laborberichten extrahiert werden.
  • Nahrungsergänzungsprotokolle — Von Ihnen erstellte Nahrungsergänzungsmittel, Dosierungen und Einnahmeprotokolle.
  • Stimmungs-Check-ins — Tägliche Stimmungs- und Stressbewertungen, die Sie erfassen.

Alle oben genannten Daten werden auf Ihrem Gerät verschlüsselt und sicher über Apple iCloud (CloudKit) synchronisiert. Ihre Daten werden in Ihrem persönlichen iCloud-Konto gespeichert und niemals auf Servern von Lawton Labs abgelegt.

3. Datenweitergabe an Dritte

OpenAI (Laborberichtverarbeitung)

Wenn Sie die Laborimportfunktion nutzen und die KI-Analyse wählen, wird Ihr Laborbericht-PDF an die API von OpenAI zur KI-gestützten Biomarkerextraktion gesendet. Das bedeutet:

  • Ihr Laborbericht-PDF — das möglicherweise Ihren Namen, Biomarkerwerte, Referenzbereiche, den Labornamen und Testdaten enthält — wird zur Verarbeitung an die Server von OpenAI übermittelt.
  • OpenAI verarbeitet diese Daten gemäß den OpenAI API-Nutzungsbedingungen und der Datenschutzrichtlinie.
  • Gemäß den Datenverwendungsrichtlinien der OpenAI API werden über die API gesendete Daten nicht zum Training der Modelle von OpenAI verwendet.
  • Die Daten werden über eine verschlüsselte HTTPS-Verbindung übertragen.
  • Es werden keine Gerätekennungen oder personenbezogene Informationen über den Berichtsinhalt hinaus übermittelt.
  • Wenn die KI-Analyse fehlschlägt, greift die App auf einen vollständig lokalen, regex-basierten Parser zurück, der keine Daten überträgt.

Sie können die Weitergabe von Labordaten an OpenAI vermeiden, indem Sie die PDF-Laborimportfunktion nicht nutzen. Die Option „Ergebnisse einfügen" verwendet einen gerätinternen Parser, der keine Daten überträgt.

4. Apple HealthKit-Daten

Gesundheitsdaten, auf die über Apple HealthKit zugegriffen wird, werden gemäß den Apple HealthKit-Richtlinien behandelt:

  • Auf Daten wird erst zugegriffen, nachdem Sie ausdrücklich die Genehmigung erteilt haben.
  • HealthKit-Daten werden nur gelesen — die App schreibt keine Daten in HealthKit.
  • HealthKit-Daten werden über Ihr persönliches iCloud-Konto mittels Apples CloudKit synchronisiert. Diese Daten werden in Ihrem privaten iCloud-Container gespeichert und niemals auf Servern von Lawton Labs abgelegt oder an Dritte weitergegeben.
  • HealthKit-Daten werden niemals für Werbe- oder Marketingzwecke verwendet.

5. Strava-Integration

Wenn Sie Ihr Strava-Konto verbinden:

  • Die Authentifizierung erfolgt über OAuth 2.0 mit PKCE — die App sieht oder speichert Ihr Strava-Passwort niemals.
  • Es werden nur Aktivitätsdaten (Art, Dauer, Distanz, Herzfrequenz) von der Strava-API abgerufen.
  • Abgerufene Aktivitätsdaten werden lokal auf Ihrem Gerät zwischengespeichert.
  • Sie können Strava jederzeit über den Profilbildschirm trennen.

6. Datenspeicherung und Sicherheit

  • Sensible Daten (API-Schlüssel, Token, Gesundheitsdaten) werden mittels expo-secure-store gespeichert, das den iOS-Schlüsselbund für verschlüsselte Speicherung nutzt.
  • In SQLite gespeicherte Gesundheitsmetriken werden mit anwendungsseitiger AES-256-GCM-Verschlüsselung geschützt. Der Verschlüsselungsschlüssel wird im iOS-Schlüsselbund aufbewahrt.
  • Ihre Daten werden über Apples CloudKit mit Ihrem persönlichen iCloud-Konto synchronisiert. Diese Daten werden in einem privaten CloudKit-Container gespeichert, der nur für Sie zugänglich ist. Keine Gesundheitsdaten werden auf von Lawton Labs betriebenen Servern gespeichert.
  • Datenexport ist im JSON-Format über den Profilbildschirm verfügbar.

7. Datenaufbewahrung und Löschung

  • Alle Daten werden auf Ihrem Gerät gespeichert und mit Ihrem persönlichen iCloud-Konto synchronisiert. Die Daten bleiben bestehen, bis Sie sie löschen.
  • Sie können jederzeit alle Daten über die Option „Alle Daten löschen" im Profilbildschirm löschen. Dadurch werden die Daten von Ihrem Gerät und aus iCloud entfernt.
  • Durch die Deinstallation der App werden alle lokal gespeicherten Daten entfernt. Daten in Ihrem iCloud-Konto können bestehen bleiben, bis sie über die App oder die iCloud-Einstellungen gelöscht werden.
  • An OpenAI zur Laboranalyse gesendete Daten werden vorübergehend verarbeitet und gemäß den Datenverwendungsrichtlinien der OpenAI API nicht von OpenAI zu Trainingszwecken aufbewahrt.

8. Datenschutz für Kinder

Die App ist nicht für die Nutzung durch Personen unter 13 Jahren bestimmt. Wir erheben wissentlich keine personenbezogenen Daten von Kindern unter 13 Jahren.

9. Analytik und Fehlerverfolgung

Die App nutzt die folgenden Dienste für Analytik und Fehlerverfolgung:

  • Amplitude — Anonyme Nutzungsanalytik, um zu verstehen, wie Funktionen verwendet werden, und die App zu verbessern. Es werden keine Gesundheitsdaten oder personenbezogene Informationen an Amplitude gesendet.
  • Sentry — Absturz- und Fehlerberichte zur Identifizierung und Behebung von Fehlern. Fehlerberichte können Gerätetyp, Betriebssystemversion und Stack-Traces enthalten, beinhalten jedoch nicht Ihre Gesundheitsdaten.

Keiner dieser Dienste erhält Ihre Gesundheitsdaten, Biomarker oder personenbezogene Gesundheitsinformationen.

10. Änderungen dieser Richtlinie

Wir können diese Datenschutzrichtlinie von Zeit zu Zeit aktualisieren. Über wesentliche Änderungen werden wir Sie über die App informieren. Die fortgesetzte Nutzung der App nach Änderungen gilt als Annahme der aktualisierten Richtlinie.

11. Ihre Rechte

Sie haben das Recht:

  • Auf alle in der App gespeicherten Daten zuzugreifen.
  • Ihre Daten in einem portablen Format (JSON) zu exportieren.
  • Alle Ihre Daten jederzeit zu löschen.
  • Jeden Drittanbieterdienst jederzeit zu trennen.
  • Funktionen, die eine Datenverarbeitung durch Dritte beinhalten (Laborimport), nicht zu nutzen.
